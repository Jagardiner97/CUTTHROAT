CUTTHROAT
=====

CUTTHROAT is a MOOSE based application named after the cutthroat trout found in the American Mountain West. CUTTHROAT uses Python code to perform heat exchanger topology optimization using kernels from the MOOSE Navier-Stokes module to perform CFD simulations. This application leverages neural networks, including Convolutional Neural Networks (CNNs) and Physics-Informed Neural Networks (PINNs), to generate heat exchanger designs and evaluate their performance.

Fork "CUTTHROAT" to create a new MOOSE-based application.

For more information see: [https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app](https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app)
